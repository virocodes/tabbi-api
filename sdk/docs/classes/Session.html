    Class Session

    Represents an active session with an isolated sandbox environment.

    Sessions are created via AgentAPI.createSession and provide methods for sending messages, accessing files, and managing the session lifecycle.

    Example

    const session = await agent.createSession();

    // Wait for the sandbox to be ready
    await session.waitForReady();

    // Send a message
    await session.sendMessage("Create a REST API", {
      onEvent: (event) => console.log(event)
    });

    // Access files
    const files = await session.listFiles("/");
    const content = await session.getFileText("/src/index.ts");

    // Cleanup
    await session.delete();

    sendMessage

    • sendMessage(content: string, options?: SendMessageOptions): Promise<Message>

      Send a message to the AI agent and stream the response.

      The agent will process the message and may use tools to read/write files, run commands, search code, and more. Events are streamed in real-time.

      Parameters

      • content: string

        The message to send

      • options: SendMessageOptions = {}

        Options including event callback and abort signal

      Returns Promise<Message>

      The final assistant message

      Throws

      AgentAPIError with code SESSION_BUSY if already processing

      Example

      const message = await session.sendMessage("Fix the auth bug", {
        onEvent: (event) => {
          switch (event.type) {
            case "message.assistant":
              process.stdout.write(event.data.content);
              break;
            case "message.tool":
              console.log(`Tool: ${event.data.toolName}`);
              break;
          }
        }
      });

      console.log("Final response:", message.content);

    waitForReady

    • waitForReady(timeoutMs?: number): Promise<void>

      Wait for the session to be ready.

      Call this after creating a session to ensure the sandbox is initialized before sending messages.

      Parameters

      • timeoutMs: number = 60000

        Maximum time to wait in milliseconds

      Returns Promise<void>

      Throws

      AgentAPIError with code TIMEOUT if timeout exceeded

      Throws

      AgentAPIError with code SESSION_ERROR if session has an error

      Example

      const session = await agent.createSession();
      await session.waitForReady(60000); // Wait up to 60 seconds
      await session.sendMessage("Hello!");
